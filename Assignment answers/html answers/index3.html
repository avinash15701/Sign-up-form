HTML tags

  1. p tag: This tag is used to write paragraph in html document.
  2. br tag: This tag is used to break the line.
  3. body tag: This tag contain the body of the html document.
  4. title tag: This tag gives the title to the content.
  5. i tag: This tag is used to italic the given text.
  6. u tag: This tag is used to underline the given text.
  7. b tag
    8. This tag is used to bold the text.
  8. nav tag
  9. head tag
